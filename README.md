# capgo-github-bot

> A simple GitHub App built with [Probot](https://github.com/probot/probot) that is a github bot to control tests in [capgo](https://github.com/Cap-go)

## Setup

```sh
# Install dependencies
bun i

# Run the bot
bun dev
```
